## SBK Forex Platform

### Overview
SBK Forex is a company profile and learning platform for a forex training institution. The platform provides educational content, sells trading materials, and offers user dashboards for students.

### Features
- Company profile and service pages
- Learning platform for forex students
- Sale of trading materials and resources
- User authentication and account management
- Personalized client dashboards

### Key Systems Implemented
- Authentication system using Django Allauth
- Payment integration via Pesapal
- Secure user data handling
- Dashboard system for user-specific content

### Tech Stack
- Backend: Django (Python)
- Frontend: HTML, CSS, JavaScript
